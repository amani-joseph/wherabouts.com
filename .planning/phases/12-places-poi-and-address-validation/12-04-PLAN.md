## Task 1 — `places` SDK resource

**Files:** `packages/sdk/src/resources/places.ts` (create),
`packages/sdk/src/resources/places.test.ts` (create)

1. Mirror `addresses.ts` structure exactly:
   - Interfaces: `PlaceSearchParams` (`q: string`, `category?: string`, `lat?: number`,
     `lng?: number`, `limit?: number`), `PlaceResult` (name, category, locality, lat, lng,
     osmType/osmId, score), `PlaceSearchResponse`, `PlacesResource`.
   - `export const createPlaces = (request: Requester): PlacesResource => ({ search: (params,
     options) => request<PlaceSearchResponse>({ method: "GET", path: "/api/v1/places/search",
     query: { q, category, lat, lng, limit }, ...options }) })`.

     `limit`).
2. Test (`places.test.ts`) mirrors `addresses.test.ts`: a `vi.fn()` requester, call
   `places.search({ q: "cafe", lat: -33.8, lng: 151.2, limit: 5, category: "food_drink" })`,
   assert `toHaveBeenCalledWith({ method: "GET", path: "/api/v1/places/search", query: {...} })`.

**Verify:** `pnpm -F @wherabouts.com/sdk test` passes the new file; types compile.

**Commit:** `feat(sdk): add places resource (places.search)`

## Task 2 — `addresses.validate` method

**Files:** `packages/sdk/src/resources/addresses.ts` (edit),
`packages/sdk/src/resources/addresses.test.ts` (edit)

1. Add to `AddressesResource`: `validate(params: ValidateParams, options?: CallOptions):
   Promise<ValidateResponse>`.
   - `ValidateParams`: structured (`street`, `locality`, `state?`, `postcode?`, `country?`) OR
     freeform (`q`) — model as a union matching the endpoint, with a `structured` discriminator
     OR a simpler `{ q?, street?, locality?, ... }` shape that maps to the query string. Keep it
     consistent with how the endpoint reads params.
   - `ValidateResponse`: `{ standardizedAddress: {...}, confidence: number, status: string,
     corrections: Array<{ field: string; from: string; to: string }> }`.
   - Implement in `createAddresses`: `validate: (params, options) =>
     request<ValidateResponse>({ method: "GET", path: "/api/v1/addresses/validate", query: {...},
     ...options })`.
2. Extend `addresses.test.ts` with a `validate` case asserting the GET request shape and path.

**Verify:** `pnpm -F @wherabouts.com/sdk test`; the new validate assertion passes; existing
addresses tests unaffected.

**Commit:** `feat(sdk): add addresses.validate method`
