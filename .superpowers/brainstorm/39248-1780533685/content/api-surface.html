🔔 Webhook Subscriptions
POST /api/v1/webhooksNEW Subscribe URL to zone entry/exit events

DEL /api/v1/webhooks/{ id}NEW Remove subscription
Flow: device location → webhook
POST /devices/ { id}/location → Worker runs ST_Contains(zone.geom, point) across all project zones → Compares result with device_zone_state.zone_ids → Detected crossings enqueued to Cloudflare Queue → Queue consumer fires POST { webhook.url} with HMAC-signed payload → Upserts device_zone_state with new zone membership.
